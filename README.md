> A [TSLint config](https://palantir.github.io/tslint/usage/tslint-json/) for [GroupBy Inc](http://groupbyinc.com/)

## Installation

```sh
npm install tslint-config-groupby --save
```

## Usage

In `tslint.json`:

```json
{
  "extends": "tslint-config-groupby"
}
```
